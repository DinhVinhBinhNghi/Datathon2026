# Modeling explanation outputs

This folder contains pre-generated explainability artifacts used in the report.

- `feature_group_importance_comparison.png`: feature-group importance comparison for Model A (2013-2018) and Model B (2019-2022).
- `feature_group_importance_comparison.csv`: values used in the figure.

The figure is based on LightGBM gain importance aggregated into business-readable feature groups. This satisfies the report explainability requirement through feature importance. No external data or test Revenue/COGS values are used.
