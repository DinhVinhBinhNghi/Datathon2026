# Archive notebooks

Đặt notebook exploration đầy đủ ở đây, ví dụ:

```text
notebooks/archive/DATATHON_exploration_full.ipynb
```

Notebook archive dùng để lưu dấu vết phân tích, không phải pipeline tái lập chính. Pipeline tái lập chính là:

```bash
python scripts/run_final_story_charts.py --data-dir data --out-dir outputs/figures/main
```
